Overview

This project is a simple chatbot built using Python.
The chatbot interacts with users through the command line and responds to basic messages like greetings or simple questions.

It uses if-else conditions to understand user input and provide predefined responses.

This project is useful for beginners who want to learn:

Basic Python programming

Loops and conditional statements

How a simple chatbot works

⚙️ Features

Interactive chatbot in the terminal

Responds to simple greetings and questions

Continuous conversation using a loop

Exit option using "bye"

No external libraries required

🧰 Technologies Used

Python
